1. 初始化npm项目
npm init 
2. 添加用户（没有账号）
npm adduser
回车弹出浏览器窗口，（需要注册https://www.npmjs.com/）
3. 登录（已有账号）
npm login
回车弹出浏览器窗口，输入邮件收到的验证码校验
4. 验证是否登录成功
npm whoami
会显示登录成功的username
5. 发布
npm publish
注意每次发布version都要更新
